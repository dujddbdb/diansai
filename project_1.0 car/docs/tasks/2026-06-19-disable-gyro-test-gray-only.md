# 关闭陀螺仪测试纯灰度循迹

**日期**: 2026-06-19

## 涉及文件
- `bsp/track_config.h`

## 修改前状态
- 实车上电后出现疯转，需要隔离判断是陀螺仪融合问题还是纯灰度/电机方向问题。
- 当前配置 `GYRO_ENABLE=1`，直线补偿和直角转弯都会受 BNO080 数据影响。

## 修改内容和原因
- 将 `GYRO_ENABLE` 改为 `0`。
- 当前固件用于纯灰度循迹测试：不初始化 BNO080，不执行陀螺仪直线阻尼，也不使用陀螺仪控制直角角度。
- 保持灰度映射和其它参数不再继续改动，方便单独观察纯灰度是否稳定。

## 测试建议
- 下载后先架空车轮，上电观察是否还疯转。
- 串口重点看 `err`、`PID`、8位灰度值和左右目标/实际转速。
- 如果纯灰度仍疯转，优先检查电机方向、编码器方向、速度环反馈符号。
- 如果纯灰度正常，再恢复 `GYRO_ENABLE=1` 单独排查陀螺仪 yaw 符号和直角 PID。
