# 修正灰度传感器通道方向

**日期**: 2026-06-19

## 涉及文件
- `bsp/grayscale.h`
- `bsp/track_config.h`

## 修改前状态
- 实车确认：传感器 CH1 在物理最左边。
- 当前 `GRAY_DIRECTION=1`，会把采集结果反向存放，导致软件 `ir_raw[0]` 变成物理最右侧。
- 直角检测使用 `ir_raw[0..2]` 和 `ir_raw[5..7]` 判断左右，因此通道反向会导致左/右直角识别反。
- 上一轮曾尝试翻转 `GYRO_YAW_DIRECTION`，但在确认 CH1 物理位置后，根因应优先修正灰度通道方向。

## 修改内容和原因
- 将 `GRAY_DIRECTION` 从 `1` 改为 `0`，使软件索引 0 对应物理最左侧 CH1。
- 将 `GYRO_YAW_DIRECTION` 恢复为 `1`，避免用陀螺仪方向补偿灰度通道映射错误。

## 测试建议
- 串口观察 8 位灰度：遮住最左侧 CH1 时，应表现为 `ir_raw[0]` 变化。
- 左直角入口应显示 `t2`，右直角入口应显示 `t1`。
- 若检测类型正确但转向仍反，再单独检查电机左右映射或陀螺仪 yaw 方向。
