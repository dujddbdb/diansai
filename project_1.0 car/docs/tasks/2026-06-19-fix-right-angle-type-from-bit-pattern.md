# 根据灰度位模式修正左右直角类型

**日期**: 2026-06-19

## 涉及文件
- `bsp/track.c`

## 修改前状态
- 实车串口位模式确认：`00001111` 表示左边有黑线。
- 当前直角检测中，`ir_raw[5..7]` 为黑时赋值 `current_type = 2`，`ir_raw[0..2]` 为黑时赋值 `current_type = 1`。
- 这与实车串口显示的左右含义不一致，会导致检测到左侧黑线时锁存成错误直角方向。

## 修改内容和原因
- 将 `ir_raw[5..7]` 黑线模式对应的 `current_type` 改为 `1`。
- 将 `ir_raw[0..2]` 黑线模式对应的 `current_type` 改为 `2`。
- 只交换类型赋值，不改注释、不改执行逻辑，避免引入额外变量。

## 测试建议
- 串口出现 `00001111` 时，应锁存为左直角对应类型。
- 再分别测试左右直角：确认 `t` 值与实际转向一致。
- 若类型正确但转向仍反，再单独检查直角执行差速或 `GYRO_YAW_DIRECTION`。
