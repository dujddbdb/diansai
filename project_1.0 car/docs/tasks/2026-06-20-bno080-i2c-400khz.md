# 2026-06-20 - BNO080 I2C 时钟改为 400kHz

## 日期
- 2026-06-20

## 涉及文件
- `bsp/bno080.c`

## 修改前状态
- BNO080 I2C1 时钟为 350kHz。
- 单次读取旋转向量包约 0.8~1.2ms，I2C 读取期间会屏蔽 TIM7/TIM11 更新中断。

## 修改内容和原因
- 将 `i2c.I2C_ClockSpeed` 从 `350000` 改为 `400000`。
- 目的：缩短 BNO08x I2C 读包时间，减少对 5ms 速度环的阻塞时间。

## 测试建议
- 下载后观察 `pk/rot/gf` 是否仍稳定增长。
- 如果 400kHz 下 BNO08x 再次停流或卡死，说明总线裕量不足，应退回 350kHz 或继续做非阻塞/DMA读取。
- 如果卡顿减轻且 BNO数据稳定，则保留 400kHz。

## 编译验证
- Keil MDK 全量 Rebuild：`0 Error(s), 4 Warning(s)`。
- 4 个 warning 仍是 `track_config.h` 末尾换行提示，已确认文件末尾存在 CRLF；本次 I2C 修改无编译错误。
