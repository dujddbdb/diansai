## 修改前状态

- 初始直角特征后只接受 7/8 路白，并在 120 ms 后取消方向锁存，不符合“先识别特征，再一直等待真正全白”的原方案。
- IMU 直角执行期间一旦 `gyro_yaw_available` 短暂变零，动作会切换到固定差速；IMU 分支末段还重新混入灰度 PID。
- 瓷砖缝造成的瞬时二值 0 会直接进入直角入口判断。
- K230 使用对角线交点，没有显式的靶面到图像单应性接口，无法在透视域标定激光安装偏差。

## 修改内容和原因

- Phase 0 对初始边缘特征确认后锁存方向；Phase 1 不再超时取消，只在检测专用结果严格等于 `0xFF` 且连续两次成立时进入动作。
- 新增直角检测专用时空滤波：黑点需连续 8 次、方向特征再连续 3 次才有效；白色恢复立即生效。正常循迹仍使用原始模拟量，不改变循迹响应。
- IMU 在动作开始时有效便锁定为 IMU-only：执行和退出只使用 yaw 与 1.2 s 安全硬超时，短时停流不切固定差速，phase 2 不再融合灰度 PID。
- K230 新增方形靶面到检测四边形的显式单应性映射。`TARGET_PLANE_U/V` 默认均为 0.5；可在归一化靶面中标定相机与激光的固定偏差，再经透视映射得到图像坐标。
- 调试 D0 页增加 `RF`，用于同时观察原始 `IR` 与直角检测滤波字节。

## 测试建议

- 手推经过瓷砖缝，确认原始 `IR` 可短暂出现 0，但 `RF` 不形成持续 `03/C0` 特征，phase 保持 0。
- 真实直角应先看到 `RF=03` 或 `RF=C0` 并进入 phase 1，随后只有 `RF=FF` 连续确认才进入 phase 2。
- 拔掉/干扰 BNO08X 数据时，已经以 IMU 模式进入的直角不得切换为固定差速；最终由恢复或 1.2 s 硬超时退出。
- 视觉先保持 `TARGET_PLANE_U/V=0.5`，若光斑固定偏左则小幅增大 U，固定偏上则小幅增大 V，每次建议调整 0.002～0.005。
